Willkommen in deinem neuen Projekt

Sage dem SuperDesign Agent was du bauen möchtest!

Nächster Schritt:

  1. Öffne SuperDesign Canvas: Ctrl/Cmd + Shift + P → SuperDesign: Open Canva
  2. Sage dem Agent: "Design [deine Idee]"
  3. Der Agent führt dich durch Layout → Theme → Animation → HTML
  4. Designs landen in .superdesign/design_iterations/
